================================================================================
								Matrix UMAT
================================================================================

1. The source code for Matrix UMAT can be foud in 'src' folder.
2. The make file can be used to create the UMAT object file.
3. Abaqus version can be specified using the attribute ABACALL. By default the
   the version used is abq2020.

	eg : make ABACALL=abq2022

4. Object file gets created in the 'bin' folder.

================================================================================
								Eshelby module - Yarn UMAT
================================================================================

1. The source code for Eshebly module can be foud in 'src' folder.
2. The make file can be used to create the Eshelby python module(Eshelby.so).
3. Abaqus version can be specified using the attribute ABACALL. By default the
   the version used is abq2020.

	eg : make ABACALL=abq2022

4. Eshelby python module gets created in the 'bin' folder.
